PocketBase 提供标准 REST API（集合/记录/文件），同时支持自定义 Routes。

## 基本约定

- JSON 响应：`{ data, meta? }`（不同端点略有差异）
- 鉴权：`Authorization: Bearer <token>` 或通过 Cookie

## 自定义 API

你可以通过 hooks 添加自定义路由，例如：

- `/api/search`
- `/api/plugins/list`
- `/api/downloads/versions`
