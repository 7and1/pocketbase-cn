## 下载与运行

推荐直接从官方 Releases 下载对应平台的 zip。

解压后运行：

```bash
./pocketbase serve
```

Admin UI 默认地址：

- `http://127.0.0.1:8090/_/`

## 生产建议（最小集）

- 反向代理（Nginx/Caddy），启用 HTTPS
- 配置备份策略（SQLite + 文件）
- 将 `pb_data/` 放到可靠磁盘（不要放临时目录）
