PocketBase 是一个「单文件可执行」的后端服务：内置 SQLite、鉴权、Admin UI、实时订阅与文件存储等能力。

## 适用场景

- 中小型项目的 API 后端（Web / App / 小程序）
- 管理后台 + 数据模型快速迭代
- 私有化部署、数据自主可控

## 核心能力

- Auth（用户名/邮箱/第三方 OAuth）
- Collections & Records（集合、字段、规则、索引）
- Realtime（订阅记录变更）
- Files（文件上传、缩略图、访问控制）
- Hooks & Routes（后端扩展：JS hooks / 自定义 API）
